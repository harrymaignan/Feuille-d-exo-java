# Feuille-d-exo-java



Exo Entraînement JS HTML
Exercice 1 — Modifier une page avec
JavaScript
Objectif
Manipuler le DOM et les événements.
Consignes
Créer une page contenant :
● un titre
● un paragraphe
● un bouton
Quand on clique sur le bouton :
● le texte du paragraphe change
● la couleur du titre change
● le fond de la page change
Contraintes
● utiliser querySelector
● utiliser addEventListener
-------------------------------------------------
Exercice 2 — Calculateur de TVA
Objectif
Utiliser les formulaires et faire des calculs.
Interface attendue
● un champ “Prix HT”
● un champ “TVA (%)”
● un bouton “Calculer”
● une zone de résultat
Fonctionnement
Afficher :
● le montant de la TVA
● le prix TTC
Exemple
HT = 80
TVA = 20
Résultat :
● TVA : 16 €
● TTC : 96 €
-------------------------------------------------
Exercice 3 — Générateur de carte de
profil
Objectif
Récupérer des informations dans un formulaire et modifier le HTML.
Interface attendue
Un formulaire avec :
● prénom
● âge
● ville
● couleur préférée
Et une carte profil vide à droite.
Fonctionnement
Quand on clique sur “Créer la carte” :
● afficher les informations dans une carte
● changer la couleur de fond de la carte selon la couleur choisie
● afficher/masquer la carte
-------------------------------------------------
Exercice 4 — Convertisseur de
température
Objectif
Utiliser les calculs et événements.
Interface attendue
● un champ de température
● un choix :
○ Celsius → Fahrenheit
○ Fahrenheit → Celsius
● un bouton “Convertir”
Formules
Exemple
20°C → 68°F
-------------------------------------------------

Exercice 6 — Jeu “Devine le nombre”
Fonctionnement
Le programme choisit un nombre aléatoire entre 1 et 100.
L’utilisateur doit le deviner.
Après chaque proposition :
● afficher :
○ “Trop grand”
○ “Trop petit”
○ “Bravo”
Contraintes
● utiliser Math.random()
● compter le nombre d’essais
● bouton recommencer
● limiter le nombre d’essais
